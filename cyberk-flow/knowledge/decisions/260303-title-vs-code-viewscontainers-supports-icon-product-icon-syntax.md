# --title VS Code viewsContainers supports $(icon) product icon syntax
**Date**: 2026-03-03

VS Code viewsContainers.activitybar icon field accepts product icon references like $(terminal) since VS Code 1.56+. No need for an SVG file for activity bar icons.
